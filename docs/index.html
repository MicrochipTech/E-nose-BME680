Key Innovation: Turbulence Relabeling

The BME680 approach eliminates the "turbulence" class entirely. Transition periods between scent jars are split at the slope reversal point — the first half keeps the fading scent's label, the second half gets the arriving scent's label. This trains the model to classify correctly during transitions (~10s detection), not just after the signal settles.
